Java的序列化

　　对象序列化是指将内存中保存的对象以二进制数据流的形式进行处理以便实现对象的传输或储存。比如要发送到服务器上、储存在文件里、存在数据库中等。但并非所有对象都可以被序列化，要序列化一定要实现java.io.Serializable的父接口，该接口没有任何方法，描述的是一种类的能力。
