基于Metasploit渗透测试框架的web应用渗透技术

　　1. 辅助模块
　　位于auxiliary下，web应用辅助扫描、漏洞查找都在此处。
　　为了方便可以使用wmap进行扫描。下面对10.10.10.129进行扫描

　　2. 渗透模块
　　metasploit针对各种web应用的渗透模块分散在module的多个文件夹下，如：expolot/unix/webapp、exploit/windows/http、exploit/multi/http等，也有针对主流CMS(Wordpress、Joomla)的漏洞，也有针对各类数据库的漏洞，同时包含渗透成功后用来操作的web shell(PHP和JSP)。有的模块没有的话，也可以自己编写加载。
　　除此之外，metasploit也提供了和主流扫描器、渗透测试软件的接口：W3AF、SQLMap、BeEF等。

文件包含和文件上传漏洞

　　文件包含(Local File Include)和文件上传(Remote File Inclusion)是文件包含攻击的两种形式。
　　
　　本地文件包含就是通过浏览器来引入web服务器上的文件，原因是因为在浏览器包含文件是没有做好过滤工作。举三个简单例子，来自DVWA。

　　
　　文件上传原理类似，也是因为对文件类型的过滤不到位，导致了可以植入木马脚本等操作。
