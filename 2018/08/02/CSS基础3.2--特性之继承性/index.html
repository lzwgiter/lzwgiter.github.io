继承性就是指被包含的元素将拥有外层元素的样式效果，继承性最典型的的应用就是在默认样式的预设上。

例：

我们来通过继承角度来考虑标记直接的树形关系：

html是根元素，所以他是所有html元素的源头。在图中每一个分支中，上层为下赠的父元素，下层为上层的子元素，如样式body{font-size: 18px; color: blue}则body中所有元素都将继承其字号设置，颜色设置。
但是，有三个继承的特殊情况：

  1. a标签的字体颜色不能继承父标签，因为a标签有默认的color: -webkit-link;颜色属性。
  2. h标签的大小也不能继承，如<h2>标签，其有默认的font-size: 1.5em;的属性。
  3. div标签的高度由内容来决定，宽度默认由父元素继承。
